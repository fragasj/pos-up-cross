# Trabalho, Aplicativo, Melhores Séries
Aplicativo desenvolvido em Xamarin.Forms para visualizar uma lista de melhores séries. 

## Descrição do Projeto
Este projeto é destinado a todos que possuem um certo interesse em visualizar as melhores séries do momento, aquelas mais bem avalidas, consumindo a API do site The Movie Database. 

## Tecnologias

* Xamarin.Form – É uma suíte de desenvolvimento de aplicativos móveis cross-platform, que utiliza C# como dificação. Com esse suíte podemos desenvolver aplicações cross-plataform ou nativas utilizando Xamarin.Android ou Xamarin.Ios.
* AutoFac – Controle e Injeção de Dependências para o .Net
* Refit – Acessar chamadas HTTP e resolução de Responses e JSON.
